# OnlineVotingSystem

A Java-based online voting system with MySQL database integration.

## Features
- Secure user login and authentication
- Candidate registration and management
- Vote casting and result tallying
- Admin dashboard for monitoring

## Technologies Used
- Java (Swing for GUI)
- MySQL
- JDBC (mysql-connector-j-9.5.0.jar)

## How to Run
1. Ensure MySQL is installed and running
2. Import the database schema from `/src/voting/schema.sql` (if available)
3. Compile and run `Main.java` from the `src/voting` folder
4. Use the GUI to interact with the system
